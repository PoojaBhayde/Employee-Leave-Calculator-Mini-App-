# HRMS Leave Management API

A backend project built using Node.js, Express, and MongoDB to manage employee leave requests.

---

## 🚀 Features

- Apply Leave (Full Day / Half Day)
- Get All Leaves
- Get Leave by ID
- Update Leave
- Delete Leave
- Input Validation using Joi

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Joi
